# Introduction to NVIDIA Isaac Sim and Isaac ROS

This chapter introduces NVIDIA Isaac Sim and Isaac ROS, powerful tools for developing AI-powered robotic applications. Isaac Sim provides a photorealistic simulation environment, while Isaac ROS provides accelerated perception and navigation capabilities.

## Learning Objectives

- Understand the NVIDIA Isaac ecosystem
- Learn about Isaac Sim's photorealistic simulation
- Explore Isaac ROS accelerated algorithms
- Set up the Isaac development environment

## NVIDIA Isaac Overview

NVIDIA Isaac is a comprehensive robotics platform that includes simulation, algorithms, and tools to accelerate the development and deployment of AI-powered robots. The platform consists of Isaac Sim for simulation and Isaac ROS for accelerated computing.
